# 🎓 캡스톤 프로젝트 소개 웹 (EC2 + Docker 배포)

본 저장소는 캡스톤 프로젝트의 단일 소개 페이지와 관련 API를 제공합니다.

---

## 1. 작품주제 (Subject)
- 제목: 네컷모아 (Nemo) – 감성적인 네컷사진 아카이브 플랫폼
- 요약: QR 코드 기반 자동 업로드와 타임라인/지도형 앨범 관리 기능을 갖춘 네컷사진 통합 관리 서비스입니다.

---

## 2. 실용적 근거 (Rationale)
- 문제: 다양한 브랜드(인생네컷, 하루필름, 포토이즘 등)에서 촬영한 네컷사진을 한곳에 정리하기 어려움.  
- 근거(수치/설문/사례):  
  - MZ세대의 72%가 1년 내 네컷사진 촬영 경험이 있으며,  
    80% 이상이 사진 정리 불편함을 호소 (자체 설문 기반).  
- 기대 가치: 
  - 자동화된 업로드와 정리로 추억 관리 효율 향상  
  - 감성 UI를 통한 회고 및 공유 기능 제공

---

## 3. 핵심기능 (Features)
- 기능 1: QR 코드 기반 네컷사진 자동 업로드  
- 기능 2: 촬영일·장소별 자동 분류 및 캘린더 타임라인  
- 기능 3: 공유 앨범 및 구독형 클라우드 저장소 (AWS S3 연동)

---

## 4. 구현환경 (Environment)
- Front-End (프론트엔드): Flutter (Dart 3.9)  
- Back-End (백엔드): Spring Boot (Java 21, JPA, JWT 인증)  
- Runtime (런타임): Docker Compose 기반  
- Deploy (배포): AWS EC2 / (선택) AWS ECS, S3 연동
